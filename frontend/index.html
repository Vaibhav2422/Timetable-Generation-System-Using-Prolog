    How to Use

    • Generate a timetable first, then return here.
    • Drag any coloured cell to a new position.
    • Green highlight = valid drop target.
    • Red highlight = constraint violation.
    • A warning modal will appear if the move violates a constraint.
    • Suggested alternatives are shown when a move is invalid.
